1 - Descrição do projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026, na categoria Programação, sub categoria Ensino Médio. O site aborda o tema “Agroforte, futuro sustentável: equilíbrio entre produção e meio ambiente”, apresentando reflexões e informações sobre [projeto foi gerenciado pelo gemini, os códigos foram sendo arrumados no projeto. esse projeto é um jeito de usar tecnologia no campo, com drones inteligentes, árvores, chuva com sensores contando tudo em tempo real ].

O projeto é destinado a estudantes, professores e comunidade em geral interessados em compreender a importância do equilíbrio entre a produção tecnológica e a preservação ambiental.

3 - Estrutura do projeto

O projeto é composto pelos seguintes arquivos e pastas:

- `index.html`: arquivo principal com a estrutura da página;
- `style.css`: arquivo responsável pela aparência e organização visual do site;
- `script.js`: arquivo responsável pelas interações e funcionalidades em JavaScript;
- `README.md`: documentação do projeto;
- `assets` ou `img`: pasta utilizada para armazenar imagens e outros recursos.
